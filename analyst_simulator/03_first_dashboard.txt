Нужен единый источник знаний - дашборд, в которых из единой базы тянутся данные.
Например, данные об аудитории - типичные метрики это DAU, WAU, MAU, CTR
DAU - SELECT COUNT_DISTINCT(user_id), time grain - day 
WAU - SELECT COUNT_DISTINCT(user_id), time grain - week
MAU - SELECT COUNT_DISTINCT(user_id), time grain - month

пример запроса

SELECT toStartOfDay(toDateTime(time)) AS __timestamp,
       count(DISTINCT user_id) AS "Daily active users"
FROM simulator_20230520.feed_actions
WHERE time >= toDateTime('2023-05-01 00:00:00')
  AND time < toDateTime('2023-06-01 00:00:00')
GROUP BY toStartOfDay(toDateTime(time))
ORDER BY "Daily active users" DESC
LIMIT 50000;
